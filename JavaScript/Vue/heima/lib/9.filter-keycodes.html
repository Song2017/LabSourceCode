Vue过滤器

全局过滤器, 所有的vue实例都可以使用

私有过滤器, 定义在vue实例内部, 只有声明的vue实例都可以使用, 名字冲突时, 私有的优先级高

定义 // data就是管道符传来的数据

Vue.filter('NameFilter',function(data){return data + '123';})

使用: {{name|namefilter}}

只能用在mustache({{}})和v-bind

按键修饰符: 注意不要使用已经被浏览器锁定的快捷键

Vue.config.keyCodes.home = 36; < input type="text" value="home" @keyup.home='press'>

获取焦点: 注意不要使用已经被浏览器锁定的快捷键

{{msg|msgFormat('duckduck')|msgAppend}}
